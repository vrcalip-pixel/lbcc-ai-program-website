# lbcc-ai-program-website

Practice website project for learning Codex Web and GitHub Pages.

## Project overview

This repository contains a simple static website concept for an "LBCC AI Program." It is designed as a beginner-friendly landing page that uses only HTML and CSS.

## Files added

- `index.html` — The main landing page with sections for the hero, overview, course pathways, student benefits, and contact/next steps.
- `styles.css` — The stylesheet that controls the page layout, colors, typography, cards, buttons, and responsive behavior.

## How to view the website

Open `index.html` in a web browser to preview the page locally. Because the project uses only static HTML and CSS, no build tools or package installation are required.
